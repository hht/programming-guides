# INPUTS — 缺则停

任一项缺失或应填 N/A 未填 → **停止写 Agent 代码**。

## 必填

| # | 项 | 验收（合格谓词） |
|---|-----|------------------|
| 1 | **用户任务** | 角色 + 可观察成功标准；非空 |
| 2 | **实现栈** | 互斥单选：□ `python` □ `typescript` |
| 3 | **模型提供商** | □ openai □ anthropic □ 其它（须写 base URL + 模型 id） |
| 3b | **模型 id** | 非空字符串（例 `gpt-4.1` / `claude-sonnet-4-20250514`） |
| 4 | **工具清单** | 表：`tool_name` \| 目的 \| 副作用(读/写/外呼) \| 超时 ms \| 是否允许 \| `fatal_on_error`(默认 no) \| **入参契约**（JSON Schema 路径 **或** 字段表：名/类型/必填）\| **观察/返回契约**（JSON Schema 路径 **或** 字段表；无结构化观察写 `N/A`） |
| 5 | **输出契约** | □ JSON Schema 路径 □ 纯文本约定；匹配器默认深等；□ 自定义匹配（须写函数名） |
| 6 | **密钥** | 环境变量名非空；DEV/PROD **成对**（同名不同值，或 `DEV_*`/`PROD_*` 前缀二选一，INPUTS 写明哪种）；**禁止**写入仓 |
| 7 | **终止条件** | 最大 turn 数（默认 **8**）+ 最大工具调用数（默认 **16**） |
| 8 | **金标集** | ≥3 条；含 ok/error/触顶；路径；若 live 烟测 N/A 写 `live_smoke: N/A — <理由>` |

## 门闸等价

| 元指南项 | 本域填法 |
|----------|----------|
| 设计 frame | `N/A` 或「CLI/API 无 UI」；有 UI 则链 ui-ux handoff |
| API/错误码 | 工具错误码表或 `N/A`（仅本地工具） |
| 环境成对 | 模型 API base + key 名；评测用 mock 提供商开关 |
| 授权 | 工具白名单 = §4；越权默认拒绝 |
| 返回信封 | **SSOT = `05`**（status/error_code/output/turns/tool_calls_count）；金标 `expected` 为其子集 |
