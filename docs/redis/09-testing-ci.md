## 单测探针（case → 期望）

| # | case | 期望 | 适用 |
|---|------|------|------|
| 1 | SET 缓存键 | TTL > 0 | 缓存 |
| 2 | 冷读 | 回源 1 次并回填 | 缓存 |
| 3 | 热读 | 不回源 | 缓存 |
| 4 | 写：源失败 | 缓存无「新值伪装成功」 | 缓存 |
| 5 | 写：只改缓存 | 拒绝/测试红（非 ephemeral） | 缓存 |
| 6 | 穿透占位 | 源调用有界 | 缓存 |
| 7 | 击穿并发 | 回源 ≪ N | 缓存 |
| 8 | 争锁 | 仅一方持有 | 锁 |
| 9 | 非持有者释放 | 锁仍在 / `LOCK_NOT_HELD` | 锁 |
| 10 | 固定窗口超限 | `RATE_LIMITED` | 限流 |
| 11 | 限流键 TTL | > 0 | 限流 |
| 12 | 会话无明文 | 仅 hash 键 | Redis 会话 |
| 13 | 无 `REDIS_URL` | 启动非 0 | 全能力 |
| 14 | Redis 读不可达（fail-closed） | `STORE_UNAVAILABLE`；不直读源 | 缓存（默认 §8b） |
| 15 | Redis 读不可达（降级） | 直读源；仅 §8b 勾降级 | 缓存+降级 |
