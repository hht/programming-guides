# INPUTS — 缺则停

任一项缺失 → 列缺口并**停止写 Redis 实现**。  
**禁止**空「其它」：凡写「其它」须带书面理由 + 可验收谓词（URL/键表/P0）。

## 必填

| # | 项 | 验收 |
|---|-----|------|
| 1 | **能力组合（至少勾一）** | □ **缓存（Cache-Aside）** □ **分布式锁** □ **限流** □ **会话存储（Redis）** — 缓存为主路径时须勾缓存；仅锁/限流/会话亦可，但 [04](./04-cache-aside-lifecycle.md) 仍为全文唯一核心路径名（无缓存时写 `N/A — 本仓无业务缓存` 并跳过 04/05 探针，见裁剪表） |
| 2 | **REDIS_URL 名** | staging/prod **成对**；变量名钉死（默认 `REDIS_URL`）；**值不入库** |
| 3 | **Redis 大版本** | 默认 **≥7**；降级须书面 + 兼容理由 |
| 4 | **应用册客户端** | □ go（**go-redis**） □ ts/node（**ioredis**） □ python（**redis-py**） □ 多册（列清单）— **禁止**同语言第二默认客户端开口 |
| 5 | **键前缀** | 业务前缀字符串钉死（例 `acme:`）；禁空前缀进生产 |
| 6 | **TTL 策略表** | 每类键：默认 TTL 秒 + 上限 + 「无 TTL」是否允许（默认**禁止**；仅 ephemeral 显式类可短 TTL） |
| 7 | **源系统** | 缓存回填/写穿的权威源（默认 **Postgres** / 对齐 [postgres](../postgres/README.md)）；或 □ **ephemeral-only**（无权威源：仅允许写 Redis；须本项勾选，见超越 a2 例外） |
| 8 | **错误码表** | 至少：`CACHE_MISS`（内部）/ `STORE_UNAVAILABLE` / `LOCK_NOT_ACQUIRED` / `LOCK_NOT_HELD` / `RATE_LIMITED` → 应用/HTTP 映射 |
| 8b | **Redis 读不可达** | **互斥钉死一**：□ **fail-closed** → 返回/映射 `STORE_UNAVAILABLE`（**默认**；不读源冒充健康） □ **降级直读权威源**（须书面：超时 ms、是否仍尝试回填、监控告警）— **禁止**「或」双开口 |
| 9 | **环境成对** | staging/prod：`APP_ENV`、`REDIS_URL`、（若会话）`SESSION_SECRET`；值不入库 |
| 10 | **应用册对接** | □ go □ fastapi □ nextjs □ react □ 多册 — 本册为 Redis 键/生命周期 SSOT |

## 若适用（按能力）

| # | 项 | 何时 |
|---|-----|------|
| 11 | **缓存空值 TTL** | 勾了缓存：默认短 TTL（建议 30–60s）钉死数字 |
| 12 | **热点键 / 单飞** | 勾了缓存：击穿策略 □ single-flight/互斥回填（**默认重试 5 次 / 总等待 200ms**；改写数字）□ 布隆（书面） |
| 13 | **锁 TTL + 持有者** | 勾了锁：锁键 TTL 秒；token = 高熵随机（≥128 bit）；**禁裸 DEL** |
| 14 | **限流窗口** | 勾了限流：默认 **固定窗口 INCR+EXPIRE**；窗口秒 + 上限 N；先进 Cell 仅当书面非默认 |
| 15 | **会话** | 勾了 Redis 会话：须同时满足 [auth INPUTS](../auth/INPUTS.md) 会话项；存 `hash(token)`；Cookie 语义以 auth 为准 |
| 16 | **DB index** | 默认 `0`；多用途分 DB 须书面表（键前缀优先于分 DB） |
