# 00 — 原则

决策序：正确性 > 可验证性 > 简洁性 > 复用 > 速度。

1. 能进库约束的不进应用碰运气。 
2. 只通过迁移改 schema。 
3. 跨表不变量同事务。 
4. RLS 默认拒绝，policy 显式允许。 
5. 密钥不入库。 
6. deletion-first。
