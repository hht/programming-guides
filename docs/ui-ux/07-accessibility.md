# 07 — 无障碍

## 不变量

- 目标：**WCAG 2.2 Level AA**（INPUTS 默认）  
- 不只靠颜色传达对错  
- 动效须尊重「减少动态」；本指南设计侧标注「可关/静态替代」  

## 检查表（必勾）

| # | 项 | 标准 |
|---|-----|------|
| 1 | 正文对比 | ≥4.5:1 |
| 2 | 大文本对比 | ≥3:1 |
| 3 | 非文字对比（图标按钮边界） | ≥3:1 |
| 4 | 触控/点击目标 | 见 `01` 表（含 macOS） |
| 5 | 焦点可见 | 清晰 focus ring；不 `outline:none` 无替代 |
| 6 | 表单标签 | 每个输入有可见 label |
| 7 | 错误 | 文本说明；不只变红 |
| 8 | 图片 | 有意义图有 alt 说明（设计标注） |
| 9 | 键盘 | Web：主路径可键盘走通（设计注明顺序） |
| 10 | 动态 | 闪烁 <3 次/秒；无限动画可停 |

## 步骤

1. 对主流程每屏跑上表。  
2. 失败项改 token/布局后重测。  
3. 将结果写入 `figma-notes.md`：`a11y: PASS` + 抽检说明。
