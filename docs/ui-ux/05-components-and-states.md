# 05 — 组件与状态（核心交付面）

> 本册核心正确性路径是 [06 Design Decision Lifecycle](./06-design-decision-lifecycle.md)；  
> **本文件钉死「可交互组件状态矩阵」交付规格**——无本矩阵的「好看图」不算完成（与 `00` / 超越 a1 一致）。

## 不变量

- 每个可交互组件在所属屏的矩阵中 **有且仅有一行**；列集合钉死（见下）；不适用列填 **`N/A`**，禁止留空。  
- 加载不得导致布局剧烈跳动：预留骨架高度或固定槽位（与 `04` 布局一致）。  
- 错误可恢复：说明原因 + 下一步；**Toast/Banner 不得作为唯一错误通道**（表单错误须贴字段）。  
- 破坏性操作：危险色 + 与主 CTA 空间分离；确认前不可逆写禁止单次误触完成。  
- 状态视觉须可被实现映射：矩阵单元格写 **可见差异一句** 或指向组件变体名 / Figma variant；禁只写「有」。

## 必做状态列（钉死）

| 列 | 含义 | 默认可 N/A 条件 |
|----|------|-----------------|
| `default` | 静止可点 / 可聚焦 | 几乎从不 N/A |
| `hover` | 指针悬停反馈 | 触控-only 平台可 N/A |
| `pressed` | 按下（含 active）；列名固定 `pressed` | 纯展示非按钮可 N/A |
| `focus` | 键盘/焦点环可见 | 不可聚焦控件须 N/A 并说明 |
| `disabled` | 不可点；对比可降但须可辨 | 无禁用语义可 N/A |
| `loading` | 进行中；防重复提交 | 无异步提交可 N/A |
| `empty` | 无数据 | 非列表/非内容槽可 N/A |
| `error` | 失败 | 无失败路径可 N/A |

列顺序与 [templates/state-matrix.md](./templates/state-matrix.md) **完全一致**；禁止增删列名（额外说明进「备注」）。

## 步骤规格（实现 / 设计自写）

1. **按屏列组件清单**  
   - 来源：INPUTS 用户任务 + `03` IA；只列本屏可交互与关键展示槽（主列表、主表单、主 CTA）。  
   - 装饰性纯图/纯分隔线不进矩阵。

2. **建矩阵文件**  
   - 路径：`design/matrices/<screen-id>.md`（`screen-id` = `02` join key）。  
   - 表头复制 templates；一行一组件。

3. **填单元格（最低信息密度）**  
   - 非 N/A 单元格至少一句：视觉或文案差异（例：`spinner 替换 label；按钮 disabled`）。  
   - `loading`：须写明 **防双提交**（disabled 或忽略二次点击）。  
   - `error`：须写明 **原因文案位置** + **下一步**（重试 / 去设置 / 联系支持等）。  
   - `empty`：须写明 **下一步**（创建 / 导入 / 澄清筛选）。

4. **主 CTA 与危险操作**  
   - 每屏主 CTA 唯一（`00`）；矩阵中主 CTA 行备注标 `primary`。  
   - 删除/断开/转账等：备注 `destructive`；与 primary 不同列视觉权重。

5. **与实现 handoff**  
   - `design/handoff.md` 须能按组件名映射到矩阵行；缺行 = 交付未完成（`09` 探针）。

6. **平台裁剪**  
   - hover 在触控 N/A 时，**不得**把唯一反馈只放在 hover；pressed/focus/disabled 仍须可辨。

## 失败分类（设计评审）

| 情况 | 行为 |
|------|------|
| 矩阵缺行 / 空单元格 | **FAIL**；补齐前不标屏完成 |
| 八列被改名/删列 | **FAIL**；恢复 templates 列集 |
| 仅有 default 图、无 loading/error | **FAIL**（适用却标 N/A 须在备注写理由） |
| Toast 作为表单唯一错误 | **FAIL**；改为字段级 + 可选汇总 |
| loading 可连点提交 | **FAIL**；须防重 |

## 单测探针（设计评审 = case → 期望）

| case | 期望 |
|------|------|
| 矩阵每行 | 八列均有值（含 N/A）；非 N/A 有可见差异说明 |
| 主 CTA 行 | 备注含 `primary`；同屏无第二 primary |
| 双击提交 | loading 态防重（文案或 disabled） |
| 表单 error | 字段级说明 + 下一步；非仅 Toast |
| 列表 empty | 有下一步，非空白页 |
| handoff 抽检 | 随机一组件能在矩阵定位到行 |
| 触控裁剪 | hover=N/A 时 pressed/focus 仍有说明 |
