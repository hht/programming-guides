# 04 — 布局与层级

## 不变量

- 视觉层级：标题 > 正文 > 辅助；主按钮最醒目 
- 对齐与间距用网格；禁无规则绝对定位堆叠 
- 首屏（营销除外）不堆统计/次要卡片抢主任务 

## 步骤规格

1. 选密度（INPUTS §5）：紧=4/8、标准=8/16、松=16/24（Web px；原生用 pt/dp 等价）。 
2. 建立 **区域**：顶栏 / 主内容 / 主操作区 / 次要；主操作区在拇指/视线可达处（移动底栏或内容末主按钮）。 
3. 表单：标签在上；错误文案在字段下；提交在表单末。 
4. 空态：插画可选；**必有**说明 + 一个下一步动作。 

## 失败分类

| 情况 | 行为 |
|------|------|
| 对比不足 | 不通过 `07` |
| 主按钮与危险按钮同色同级 | 必须区分 |
